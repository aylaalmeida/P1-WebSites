# Projeto Web - Queimadas

## Descrição

Este projeto tem como objetivo apresentar informações sobre queimadas de forma clara e objetiva, por meio de uma interface simples e de fácil navegação.

O site é composto por cinco páginas, organizadas para facilitar o acesso ao conteúdo e melhorar a experiência do usuário.

## Funcionalidades

* Navegação intuitiva entre páginas
* Estrutura organizada para apresentação de informações
* Conteúdo distribuído em múltiplas páginas
* Elementos visuais para apoio à compreensão do tema

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## Contexto Acadêmico

Projeto desenvolvido como avaliação acadêmica, com foco na aplicação de conceitos de desenvolvimento web e organização de conteúdo digital.
